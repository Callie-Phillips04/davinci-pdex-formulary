Name Definition
Drug Alternatives {% capture md_text %}Generic or Branded Drugs included within the CoveragePlan's formulary that could be prescribed as alternatives to the current FormularyDrug.{% endcapture %}{{ md_text | markdownify }}
Drug Tier Definition {% capture md_text %}A description of the drug tiers used by the formulary and how those tiers implement copay and coinsurance amounts. Drug tiers do not have any inherent meaning that is consistent across all formularies. Rather, each tier is defined using this element.{% endcapture %}{{ md_text | markdownify }}

Marketing URL {% capture md_text %}The URL that goes directly to the plan brochure for the specific standard plan or plan variation{% endcapture %}{{ md_text | markdownify }}
Network {% capture md_text %}Array of Networks within a plan.{% endcapture %}{{ md_text | markdownify }}
Plan ID {% capture md_text %}Unique, 14-character, HIOS-generated Plan ID number. (Plan IDs must be unique, even across different markets.){% endcapture %}{{ md_text | markdownify }}
